广播（broadcasting）

指的是不同形状的数组之间的算术运算的执行方式。

广播的原则：

两个数组从后向前对比，每个维度数组兼容，当：

  1. 数值相等
  2. 或其中一个数值为1
    如果较短的数组维度遍历完毕，则可以广播，否则抛出ValueError: frames are not aligned异常

当比较的任何一个维度为1时，则使用另一个。换句话说，大小为1的维被拉伸或“复制”以匹配另一维。
