模糊数学方法及其应用

Chapter 1 模糊集的基本概念

介绍模糊集的基本概念、运算法则、基本定理

概述

将模糊概念定量化、数学化——一种研究和处理模糊概念的数学方法。
量：确定性——经典数学：微分法、微分方程
不确定性——随机性(随机数学)：概率分布方法、马尔可夫链
——模糊性(模糊数学)

模糊理论的数学基础

幂集：集合A的所有子集组成的集合称为A的幂集
直积：X，Y是两个集合，由X的元素与Y的元素配对成的全体序对组成一个集合，称为X与Y的直积(也称笛卡尔积)
集合A的特征函数：特征函数是一种特殊的映射，值域为{0,1}; x属于A，值为1；x不属于A, 值为0.
集合和特征函数互相决定；集合的性质完全由其特征函数决定。
点-集映射：将点映射为集合。(一对多)
集合变换：将集合变为集合。
经典扩张原理
二元关系：集合X, Y: X 和Y的直积的子集R称为X到Y的二元关系。
等价关系：满足自反性、对称性、传递性的二元关系。
相似关系：满足自反性、对称性。
将经典集合A的特征函数的值域由{0,1}推广到[0,1]
经典集合的特征函数就扩展为模糊集合的隶属函数.

(1) 扎德表示法

(3) 向量表示法
截集
支集：A = { x | A(x) > 0 }
核：Ker A = {x | A(x) = 1 }
边界：Bd A = { x | 0< A(x) <1}

Chapter 3 模糊模型识别

模型识别：事先已知若干标准模型(标准模块库)、有待识别的对象。
模糊模型识别：在模型识别中，模型是模糊的
模糊模型识别的两种基本方法：最大隶属度原则和择近原则方法。
通过待识别对象是否是模糊的，可以将模糊模型识别分为两类。

第一类模糊模型识别

模糊向量、最大隶属度原则

第二类模糊模型识别

待识别对象也是一个模糊集，即一个模糊集对标准集的识别问题。
涉及到两个模糊集的贴近程度问题。
单独使用内积或者外积还不能完全刻画两个模糊集A，B之间的贴近程度。
格贴近度 = 1/2 * [内积 + 1 - 外积]
贴近度的改进：贴近度的公理化定义
